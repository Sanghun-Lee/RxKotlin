# chapter 01

## 리액티브 프로그래밍 소개

전통적으로 리스트를 확인하기 위해선 `iteration`을 사용하였다.

하지만, iteration코드의 경우, 데이터를 받아올 때 까지 기다려야 한다(스레드가 blocking 상태이다).

단순하게 생각하였을 때 데이터가 준비 될 때 까지 기다리는 것이 아닌! **데이터가 사용 가능 해 졌을 때 데이터가 프로그램에게 알려주는 것**이 더 좋다.
